# Number guessing game
a game that lets you guess numbers from 1 to 100 
you have 10 attempts
# the game mechanics
the game is made using python tkinter
the game uses a main app class and has many mechanics
# danger level mecanic
as you aproach the max guesses the color of the text morphs from green to red
they are in this order ["#00FF00", "#33FF00", "#66FF00", "#99FF00", "#CCFF00", "#FFFF00", "#FFCC00", "#FF9900", "#FF6600", "#FF3300", "#FF0000"]

# reset mechanic
if you win or lose the game
you can reset using the reset button instead of closing and opening the game again
when you reset it changes the number you have to guess and re-enables the ability to guess the number

# automatic screen adjustment
the screen always adjusts for your window being 2/3 times the screen and the program is scaleable
